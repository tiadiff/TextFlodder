A minimal and elegant automation utility for Windows, ported from the original macOS version.

## Features
- **Compact Dark UI**: Matches the macOS aesthetic.
- **Global Hotkey**: Press `Ctrl + Shift + Alt + F` to start/stop flooding from anywhere.
- **Sequential Flooding**: Ensures long text messages are sent correctly.

## How to Run

1. **Prerequisites**: Windows 10 or 11 with .NET Desktop Runtime 8.0 installed (usually included in modern updates, or download from Microsoft).
2. **Launch**: Double-click `Flodder.exe`.
3. **Use**:
   - Type message.
   - Set delay (ms).
   - Click START or use `Ctrl + Shift + Alt + F`.

## Troubleshooting
- If it doesn't open, install [.NET Desktop Runtime 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0).
